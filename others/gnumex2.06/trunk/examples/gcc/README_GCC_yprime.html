testGCC.m: MATLAB Script that test GCC with MATLAB's default yprime.c.

Instructions

Configure gnumex for GCC using the Documentation on gnumex usage and the lines typed into the MATLAB command line below. Your answer should be the same.

>>gnumex
>>testGCC
>>yprime(1, 1:4)
ans =

            2       8.9685            4      -1.0947
