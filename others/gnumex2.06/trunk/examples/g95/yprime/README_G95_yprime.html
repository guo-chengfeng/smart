Files

mexinterface.f90: Fortran interface module.
yprime95.f90: Module containing mexfunction gateway and yprime SUBROUTINE.

Instructions

configure gnumex for g95 using the Documentation on gnumex usage and the lines typed into the MATLAB command line below. Your answer should be the same.

>>gnumex
>>mex -c mexinterface.f90
>>mex yprime95.f90
>>yprime95(1, 1:4)
ans =

            2       8.9685            4      -1.0947
